Q1- Value at risk or VAR is a measure of the risk of loss for investments and is widely used by financial institutions. Mathematically, Value at Risk at a given probability q estimates the loss/gain of a stock/portfolio/investment during a fixed time, such that the probability to suffer a higher loss is no more than q.

To describe the price of an asset over time, we will use a simplified version of a popular stochastic process model called Geometric Brownian motion. The Geometric Brownian Motion (GBM) describing the investment price follows a random walk with drift. That is, each movement from day to day is random and is independent of past price movements, and the movements follow the same distribution and may, on average, drift upwards or downwards.

We define our model to be St=St−1×(μ+σϵt), where: - St is the price at day t - μ is the relative expected daily change - σ is the standard deviations of the daily change - ϵt is the random variable determining the actual change in price at day t.

D- Write a function called gbm.simulate that accepts initial_price,n_days, mu,sigma and outputs the Geometric Brownian Motion prices vector (including the initial starting price) along these days. For this, assume ϵ∼N(0,1). Once done, test your function using f.stock.return(9000,20,0.005,0.008).

# create function gbm.simulate that accepts initial_price,n_days, mu,sigma and outputs the Geometric Brownian Motion prices vector. 
nasdaq_ <- as.data.frame(nasdaq_)
gbm.simulate <- function(initial_price,n_days, mu,sigma){ #notice we will allways one number more - the initial price, so the function will return n_days + 1 length vector
  s <- rep(0, (length(n_days)-1))
  s[1] <- initial_price
  mu <- mean_relative_daily
  sigma <- sd_relative_daily
  epsilon <- rnorm(1) 
  for (i in 1:(n_days-1)){
     s[i+1]=s[i]*(1+mu+sigma*epsilon)
  }
  return(s)
}
